Installation

To work with the django-rbaca application, you need to install it into your Python environment. Follow the steps below to set up the application and its dependencies.

Local building and installation

After cloning, open your terminal and navigate to the root directory of the django-rbaca project:

cd path/to/django-rbaca

Replace path/to/django-rbaca with the actual path to the root directory of the project.

First activate your python environment and ensure that poetry is installed:

pip install poetry

To install the production dependencies and the package itself, run:

poetry install

This command will automatically install the necessary packages and dependencies for the django-rbaca application. For installing the development dependencies, run:

poetry install --with dev

Congratulations! You’ve now set up django-rbaca for development and can start exploring its features and capabilities.

Installation with Pip

pip install django-rbaca
